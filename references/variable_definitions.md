# 4.2 Variable Definition and Data Check

## 1. Categorical Variable: SadOrHopeless
- **What it measures:** Whether the student felt sad or hopeless for 2+ weeks in a row.
- **Valid codes:** 1 (Yes), 2 (No).
- **Recoding rule:** Code 1 is defined as "Success" (1), others are "Failure" (0).
- **Missing values:** All NaNs are removed to ensure a clean sample.

## 2. Continuous Variable: BMIPCT
- **What it measures:** BMI percentile (0-100).
- **Valid codes:** Numerical values.
- **Missing values:** Removed observations with missing BMI data.
